My attempt at recreating the e-card game from Kaiji manga/anime with websockets.
This will be quite bumpy as I figure out how to make Javascript and Python work
together.

All original code is public domain; all dependencies belong to their original authors

dependencies:
* [flask](https://github.com/pallets/flask)
* [flask-socket](https://github.com/miguelgrinberg/Flask-SocketIO)
* [socket.io](https://socket.io/) 4.7.5
* [jquery](https://jquery.com/) 3.7.1
* [jquery-cookie](https://github.com/carhartl/jquery-cookie)
